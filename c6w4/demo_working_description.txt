1. Сначала требуется создать новую рабочую среду с помощью conda или virtualenv; необходимые для корректной работы версии Python и пакетов в приложенном файле package_version.txt.
2. Запустить файл flask_server.py
3. С помощью Postman или ThunderClient послать GET-запрос с текстом отзыва (body >> text) по адресу http://127.0.0.1:5000/reviews. 
Запрос может быть в том числе пустым. Если в тексте есть кавычки или апострофы, то они должны предваряться обратным слешем (пример есть в документе reviews_example.txt).
Примеры как это делается можно посмотреть в приложенных скриншотах (три файла demo_*.png).

P.S. Также возможно изучить построение модели в приложенном ноутбуке movie_reviews_model.ipynb, однако она не особо примечательна с математической точки зрения - это TfidfClassifier и LogisticRegression со дефолтными настройками, обученные на корпусе movie-reviews из библиотеки nltk.
