1. Сначала требуется создать новую рабочую среду с помощью conda или virtualenv; необходимые для корректной работы версии Python и пакетов в приложенном файле package_version.txt.
2. Запустить файл server.py
3. С помощью Postman или ThunderClient послать GET-запрос с текстом отзыва (body >> text) по адресу http://127.0.0.1:5000/phone. 
Запрос может быть в том числе пустым, тогда сервер выдаст сообщение с предложением ввести отзыв (demo7.png)
Примеры как это делается можно посмотреть в приложенных скриншотах (файлы demo1.png - demo6.png); также можно запустить исполнение файла test.py, где на паре десятков разнообразных примеров можно посмотреть отзывы (скриншот demo_test.png)
